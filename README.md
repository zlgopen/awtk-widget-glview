# awtk-widget-glview

基于 TinyGL 实现 GLView，可以实现基本的3D绘图功能。

## 准备

1. 获取 awtk 并编译

```
git clone https://github.com/zlgopen/awtk.git
cd awtk; scons; cd -
```

2. 获取 awtk-widget-glview 并编译
```
git clone https://github.com/zlgopen/awtk-widget-glview.git
cd awtk-widget-glview; scons
```

## 运行

```
./bin/demo_glview
```
